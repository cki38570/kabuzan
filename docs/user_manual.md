# 📖 Kabuzan-AI ユーザーマニュアル

Kabuzan-AI を最大限に活用するためのガイドです。本アプリは、テクニカル分析と最新のAI（Google Gemini）を統合し、あなたの投資判断を強力にサポートします。

## 1. クイックスタート
1. **銘柄検索**: 画面トップの検索バーに銘柄コード（例: `7203`）を入力します。
2. **分析確認**: 入力後、自動的に解析が始まり、チャートとAI分析が表示されます。
3. **ウォッチリスト**: 頻繁にチェックする銘柄はサイドバーからリストに追加しておくと便利です。

## 2. AI分析機能の詳細解説
Kabuzan-AI の最も強力な機能です。

### 🤖 深層思考プロセス (Self-Reflection)
単に AI が意見を述べるのではなく、プロンプト内で **2 人の仮想エキスパート**を戦わせています：
- **強気派 (Bull)**: 上昇の可能性、ポジティブな材料、テクニカルの好転を重視。
- **弱気派 (Bear)**: 下落リスク、需給の悪化、上値抵抗線、マクロリスクを指摘。

この両者の激しい議論を「シニア戦略アナリスト（AI）」が統合し、中立的かつ客観的な**最終判断（総合スコア）**を算出します。

### 📊 総合スコア（100点満点）の見方
- **テクニカル (60点)**: 移動平均線の並び、RSI、ボリンジャーバンドなど客観的データ。
- **センチメント (40点)**: 最新ニュースの感情分析、市場の期待値。

> [!TIP]
> スコアが 80点以上の場合は「BUY ENTRY」、40点以下の場合は「NO TRADE / SELL」のように、色分けとゲージで直感的に判断できます。

## 3. 主要機能の使い方

### 📈 マルチタイムフレーム・チャート
- **日足 (Daily)**: 中短期の売買タイミング。5/25/75日移動平均線を表示。
- **週足 (Weekly)**: 長期的なトレンドの把握。13/26/52週移動平均線を表示。
- **チャートエンジン**: インタラクティブな `Plotly` と、プロ仕様の `TradingView (TV)` 形式を切り替え可能です。

### 🔍 市場スクリーニング（市場スキャン）
「今、チャンスの銘柄」を探す機能です。
- 日経225、グロース市場、注目株（7203等）をワンクリックでスキャン。
- RSIの売られすぎ（30以下）や、ゴールデンクロスなどのシグナルを自動で抽出します。

### 💰 ポートフォリオとポジション計算機
- **管理**: 取得単価と株数を入力すると、現在の損益をリアルタイムで追跡。
- **計算機**: 自分の運用資金と「最大何円まで負けていいか」を入力するだけで、推奨の購入株数を自動計算します。

## 4. LINE通知の設定
重要なチャンスを逃さないための機能です。
1. サイドバーの「通知設定」から LINE Notify のトークンを入力。
2. RSIが極端な数値になった際などに、あなたのスマホへ直接通知が飛びます。

## 5. よくある質問 (FAQ)

### Q. AI分析の結果が「モック（ダミー）」と表示される
A. Google Gemini API の無料枠制限や、APIキーの未設定時に発生します。しばらく時間を置くか、管理者へお問い合わせください。

### Q. データが最新ではない気がする
A. 本アプリはパフォーマンス向上のためキャッシュを利用しています。最新データが必要な場合は、ページをリロードするか、少し時間をおいてから再度検索してください。

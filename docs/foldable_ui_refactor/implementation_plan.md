# フォルダブル（Pixel Fold）特化型UI実装計画

Google Pixel Foldのようなデバイスの特性（「閉じた時の細長い画面」と「開いた時の正方形に近い大画面」）を最大限に活かし、市場を俯瞰しながら詳細を深掘りできるプロ仕様のUIへ進化させます。

## デザインコンセプト：Adaptive Dual-Pane
- **Folded (閉じた時)**: これまでのモバイルファーストUIを維持。シングルカラムで縦にスクロール。
- **Unfolded (開いた時)**: 2つのペインを左右に配置。左に「意思決定サマリー（Dashboard/Strategy）」、右に「視覚的詳細（Chart/Details）」を常時表示し、画面遷移の手間を省く。

## 変更内容

### 1. レスポンシブ・サイドバイサイド構造
[app.py](file:///c:/Users/GORO/Desktop/kabuzan/app.py) のメインセクションを `st.columns([1.2, 1])` のような比率で分割します。
- **左ペイン**: AI判定、サマリーダッシュボード、トレードプラン、バックテスト結果。
- **右ペイン**: チャート（日足・週足）、詳細テクニカル、ニュース、決算詳細。
- **Streamlitの特性**: 画面幅が狭い場合は自動的に2ペイン目が1ペイン目の下に移動するため、特別なメディアクエリなしでフォルダブルに対応可能です。

### 2. コンポーネントの洗練
- **Glassmorphism 2.0**: 背景の透過度とぼかしを調整し、Pixel Foldの有機ELディスプレイに映える、より高級感のあるダークモード・デザインを採用。
- **情報密度の最適化**: 大画面時には情報を「広げる」のではなく、適切な余白（Gutter）を設けて視線の移動をスムーズにします。

# 今後の改善計画 (Implementation Plan)

現状分析に基づき、Kabuzanをさらに進化させるためのロードマップを提案します。

## 1. リアルタイム性の強化
- **目標**: 市場変動への即応性を高める。
- **内容**: `yfinance` 以外の軽量データソースの検討、または `app.py` の更新間隔の最適化。

## 2. 分析精度の向上 (Advanced Signals)
- **目標**: 売買タイミングをより鋭くする。
- **内容**: 
  - `modules/analysis.py` への「出来高急増（Volume Spike）」検知の追加。
  - `modules/patterns.py` の検知パターンの拡充。

## 3. UXと通知の高度化
- **目標**: ユーザーの手を煩わせずに重要な判断を届ける。
- **内容**:
  - LINE通知での「AI一言ニュースコメント」の実装（`modules/notifications.py`）。
  - ポートフォリオ銘柄の自動定期診断機能（バッチ処理）。

---
**本計画は現状分析の結果に基づいた提案であり、具体的な実装はユーザーのフィードバックを経て進めます。**
